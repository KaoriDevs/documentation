# $unban

This function unbans the user from the guild

```
$unban[user ID;reason (optional)]
```

{% hint style="info" %}
Reason will show up in audit logs
{% endhint %}

```javascript
bot.command({
name: "unban",
code: `Unbanned the user with the ID of 535566311942651924
$unban[535566311942651924]`
})
```
