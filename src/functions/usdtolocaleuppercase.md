# $toLocaleUppercase

This function uppercases the first character in each argument/word in \<text>

```javascript
$toLocaleUppercase[text]
```

```javascript
bot.command({
name: "toLocaleUppercase",
code: `$toLocaleUppercase[aoijs is awesome]` //Returns: Aoijs Is Awesome
})
```
